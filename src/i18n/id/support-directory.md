# Referensi Referensi Referensi 

Tujuan halaman ini adalah untuk menyediakan sebuah direktori untuk kontak pendukung. Semua orang yang saat ini bekerja dengan proyek Decred memiliki nama pengguna Slack dan/atau IRC yang terdaftar.

## Bergabunglah dengan kami di Slack

Bergabunglah dengan kami di Slack untuk mendapatkan dukungan. Anda dapat menerima undangan ke Decred Slack dengan mengunjungi [https://slack.decred.org](https://slack.decred.org). Setelah Anda mendaftar, Anda dapat bergabung menggunakan browser web Anda dengan mengunjungi [https://decred.slack.com](https://decred.slack.com) atau dengan mendownload salah satu aplikasi desktop atau mobile Slack.

## IRC Informasi:

Mayoritas pengembang tersedia di kanal Freenode IRC #decred. Bergabunglah dengan kami di klien IRC Anda sendiri atau webchat Freenode [di sini](https://webchat.freenode.net/?channels=decred&uio=d4). Ada juga jembatan antara #general Slack dan saluran #decred di IRC.
