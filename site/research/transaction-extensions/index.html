Transactions Extensions¶

Transaction expiry has been added, which allows one to prune transactions from the memory pool if the blockchain has reached a certain height1. Previously the only way to remove a transaction from the mempool was to double spend it.

  1. ByteCoin. 2010. Need OP_BLOCKNUMBER to allow “time” limited transactions. ↩
